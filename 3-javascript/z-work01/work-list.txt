들어가야 하는 내용

  연락처 추가: 이름, 전화번호, 이메일정보 입력. '추가' 버튼 클릭 테이블 목록 가장 앞쪽에 추가

  연락처 목록: 테이블셀은 이름, 연락처, 이메일, 작업이 존재해야 하며, 입력목록이 보여야함

  연락처 삭제: '작업' 셀에 삭제버튼이 존재. 버튼 클릭 시에 해당 행이 삭제되어야 함

bootstrap 5 UI 프레임워크를 이용하여 작성

  입력폼: Form control 컴포넌트 사용 작성
  버튼: Button 컴포넌트 사용 작성
	  추가 버튼은 primary 스타일 적용
	  삭제버튼 secondary, small 스타일 적용
  테이블: Table 컴포넌트 사용 작성
	  striped rows 스타일 적용
