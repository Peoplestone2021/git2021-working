떡볶이

배달떡볶이 퀄리티 그대로, 직접 만들어봅시다.
죠스와 엽떡의 레시피를 혼합한 버전입니다.

재료

  • 양념: 다진마늘1T, 고추가루2T, 설탕1T, 쇠고기다시다1t, 라면스프1t,
    카레가루1t, 고추장1t, 물엿2T, 후추가루 적당량, 물 350ml
  • 떡10개
  • 어묵2장
  • 양배추 적당량
  • 비엔나소세지5개
  • 청양고추2개

조리방법

  1. 넉넉한 크기의 냄비를 준비하고, 분량의 양념을 잘 저어가며 4분간 끓입니다.
  2. 양념이 고루 섞여 잘 끓는지 확인 후 떡, 어묵, 소세지를 넣습니다.
  3. 숟가락이나 주걱 등으로 잘 저어가며 익힙니다.
  4. 국물이 졸아들어, 바닥을 긁었을 때 냄비 바닥이 살짝 보일 정도가 되면
    양배추와 청양고추를 넣습니다.
  5. 만약 치즈를 추가 하려면, 해동된 상태의 치즈를 위에 얹고 뚜껑을 덮습니다.
    약불로 1분 익힌 뒤 뚜껑을 덮은 채 조금 더 녹여줍니다.

두부김치

막걸리나 청주와 잘 어울리는 안주입니다.
백종원 레시피를 참고했습니다.

재료

  • 양념: 설탕1t, 고추가루1t, 후추 적당량, 맛술1t, 간장1T, 굴소스1t, 고추장1t,
    다진마늘1t
  • 돼지 목심 100g
  • 김치 200g
  • 두부 반모
  • 청양고추 2개
  • 양파 50g
  • 파 적당량

조리방법

  1. 두부 삶을 물을 끓입니다.
  2. 끓는 물에 두부를 넣고 7분 가량 익힙니다.
    꺼내기 전 참기름을 살짝 두르면 풍미가 좋습니다.
  3. 팬에 고기를 볶습니다.
  4. 기름이 필요하면 조금 둘러도 좋습니다. 파와 마늘을 함께 볶습니다.
  5. 설탕을 먼저 넣고 같이 볶습니다.
  6. 나머지 양념을 넣습니다.
    재료가 타지 않도록 물을 숟가락으로 조금씩 넣어주며 볶습니다.
  7. 두부와 속을 그릇에 담아냅니다.

비빔국수

여름에 후다닥 해먹기 좋습니다.
아하부장 레시피를 참고했습니다.

재료

  • 양념: 양파가루2t, 마늘가루2t, 고추가루1T, 설탕3T, 굴소스2t, 진간장1T, 식초4T,
    고추장3T - 필요한 양만큼 비율을 맞춰 배수로 준비하는 것이 좋습니다.
    국수의 양에 맞춰서 양념장을 첨가해야하기 때문입니다. 냉장보관 합니다.
  • 적당량의 소면
  • 소금1T
  • 김치 50g
  • 참기름
  • 김가루

조리방법

  1. 넉넉한 냄비에 소금1T와 물을 넣고 끓입니다.
  2. 끓는 물에 소면을 넣습니다.
  3. 거품이 일기 시작하면 넘치지 않도록 불을 조절합니다.
    약한 불로도 면은 충분히 익힐 수 있습니다.
  4. 면의 종류에 따라 익히는 시간은 모두 다르니 젓가락으로 조금씩 건져서
    물에 헹궈 직접 먹어봅니다.
    쫄깃한 느낌이 들면 덜익은 상태입니다. 잘 익혀야 찬물에 면을 식혀도 밀가루맛이 안납니다.
  5. 면을 채반에 받치고, 흐르는 물에 비벼서 씻고, 손바닥으로 잘 눌러서 물기를 짭니다.
  6. 비벼서 씻고 짜내는 과정을 3회 반복합니다.
  7. 그릇에 면을 담고, 참기름, 김치, 김가루를 얹습니다.
  8. 양념은 3T 정도만 넣은 후 색깔을 보며 더 넣어 조절합니다.
